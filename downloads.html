In the box

What's included

A single download bootstraps the entire platform - no separate components to assemble.

  • Electron desktop UI
    HITL dashboard, alert triage, agent monitor. Native window; no browser required.
  • FastAPI backend
    REST + WebSocket API serving the pipeline, auth, and audit log.
  • Docker Compose stack
    Kafka, Redis, PostgreSQL, vLLM, ChromaDB: one command to start the full stack.
  • Wazuh bridge
    Connector that streams Wazuh detections into the Kafka backbone.
  • LangGraph agents
    All eight pipeline agents with default prompts and playbook templates.
