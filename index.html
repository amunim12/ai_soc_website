LangGraph Multi-Agent Pipeline

Eight specialized agents (Ingest, Triage, Enrich, Correlate, Decide, Respond, Audit, Escalate) coordinate through a stateful directed graph. Each node has isolated memory, deterministic handoffs, and a hard rollback path if confidence degrades mid-chain.

// 8 nodes · stateful graph · deterministic handoffs

Human-in-the-Loop Oversight

Below the 85% confidence threshold, analysts get a structured approval queue with full reasoning traces before any action fires.

// HITL gate · < 0.85

RAG-Powered Playbook Generation

ChromaDB retrieval grounds every response in your own runbooks and prior incidents: generated playbooks, not hallucinated ones.

// ChromaDB · grounded

Wazuh SIEM Integration

Native bridge ingests Wazuh detections in real time, normalizing alerts into the pipeline at over 1,100 events per second.

// Wazuh 4.7+ · native

MITRE ATT&CK Enrichment

Every alert is mapped to tactics and techniques, with adversary context and recommended mitigations attached automatically.

// TTP mapping · auto-tagged

Shuffle SOAR Automation

Approved decisions trigger Shuffle workflows: isolate hosts, rotate credentials, block IPs, executed and logged end to end.

// SOAR workflows · full audit
